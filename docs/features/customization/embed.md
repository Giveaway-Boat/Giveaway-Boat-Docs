# Embed Customization

You can customize the embed sent by the bot in giveaways.

## Options

### Color

The default color of the embed.

- Dashboard: Color option
- Slash Command: color option
- Prefix Command: `--color` or `-co`

![Color](/assets/features/customization/embed/color.png)

### End Color

The color when the giveaway ends.

- Dashboard: End Color option
- Slash Command: end-color option
- Prefix Command: `--end-color` or `-eco`

![End Color](/assets/features/customization/embed/end-color.png)

### Image

The large image seen above the footer.

- Dashboard: Image option
- Slash Command: image option
- Prefix Command: `--image` or `-i`

![Image](/assets/features/customization/embed/image.png)

### Thumbnail

The small image seen at the top right of the embed.

- Dashboard: Thumbnail option
- Slash Command: thumbnail option
- Prefix Command: `--thumbnail` or `-t`

![Thumbnail](/assets/features/customization/embed/thumbnail.png)
